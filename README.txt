RL TRACKER - Portable Edition v2 (FAST)
========================================

HOW TO USE:
  1. Double-click RL-Tracker.exe
  2. Browser opens -> http://localhost:3010
  3. Click bottom-right gear icon -> enter your RL name
  4. RL Stats API Config -> "Auto-Create"
  5. RESTART Rocket League
  6. Play! Matches track automatically.

TO EXIT: Click the red power button (bottom-right)
